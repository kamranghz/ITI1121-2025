# ITI1121-2025: Lecture 23

Welcome to the **Lecture 23** repository for **ITI1121 – Introduction to Computing II (Winter 2025)**. This lecture focuses on **Binary Search Trees (BSTs)**, including insertion, deletion (iterative and recursive), and traversal operations.

📁 GitHub Repo: [https://github.com/kamranghz/ITI1121-2025/tree/main/Lecture-23](https://github.com/kamranghz/ITI1121-2025/tree/main/Lecture-23)

---

## 📂 Repository Structure

### 🔹 Root Directory

| File Name                               | Description |
|----------------------------------------|-------------|
| `BinarySearchTreeInOrderDemo.java`     | In-order traversal of a BST. |
| `BinarySearchTreePostOrderDemo.java`   | Post-order traversal demonstration. |
| `BinarySearchTreePreOrderDemo.java`    | Pre-order traversal demonstration. |
| `BinarySearchTreeTake2.java`           | Refined or alternative implementation of a BST. |
| `BooleanExpressionTreeDemo.java`       | Demonstrates a tree-based evaluation of Boolean expressions. |
| `BooleanValueBSTDemo.java`             | Uses BST to represent boolean values. |
| `SimpleBinarySearchTreeDemo.java`      | Basic example of BST structure and operations. |
| `PreOrderIteratorBinarySearchTree.java`| Custom iterator for pre-order traversal of a BST. |

---

### 📁 Add Element

| File Name           | Description |
|---------------------|-------------|
| `BinarySearchTree.java` | Core BST structure supporting element insertion. |
| `BSTTest.java`          | Test class to verify adding elements to the BST. |

---

### 📁 Remove Element

| File Name                                 | Description |
|------------------------------------------|-------------|
| `RemoveBinarySearchTree.java`            | Standard element removal from a BST. |
| `RecursiveRemoveBinarySearchTree.java`   | Recursive approach to node deletion. |
| `AlternativeRemoveBinarySearchTree.java` | Alternative strategy for removing elements in a BST. |

---

## 🚀 Getting Started

### Prerequisites
- Java 11+
- Any Java IDE (e.g., IntelliJ IDEA, Eclipse) or terminal with `javac` and `java`

### Compilation and Run Example
```bash
cd Lecture-23/Add Element
javac BinarySearchTree.java BSTTest.java
java BSTTest
```

## 📖 Topics Covered
- 📌 BST insertion and structure
- 📌 Recursive and iterative deletion
- 📌 Traversal methods: in-order, pre-order, post-order
- 📌 Expression evaluation using trees
- 📌 BST iterator implementation
